# 🏆 BiblioUniColombo - Nix.js Library System

[![Version](https://img.shields.io/badge/version-1.7.3-blue.svg)](https://www.npmjs.com/package/@deijose/nix-js)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Una aplicación interactiva para la gestión de préstamos de libros, desarrollada como proyecto para la asignatura de **Análisis de Sistemas 2**. Utiliza **Nix.js**, el micro-framework JavaScript ultra-ligero enfocado en reactividad nativa y simplicidad extrema.

## ❄️ ¿Qué es Nix.js?

Nix.js es un micro-framework de ultra-bajo peso (~10KB gzipped) diseñado para desarrolladores que buscan:
- **Cero dependencias**: Sin sobrecarga innecesaria.
- **Reactividad Nativa**: Basado en el sistema de *Signals*.
- **Sin Build Steps**: Compatible directamente con ES Modules en el navegador (aunque este proyecto usa Vite para optimización).

## 🚀 Sobre este Proyecto

Este proyecto implementa una plataforma de biblioteca digital que permite a los estudiantes y administradores gestionar el ciclo de vida de los libros:

- **🔐 Autenticación**: Sistema de acceso seguro para usuarios y administradores.
- **📚 Catálogo Digital**: Exploración interactiva de la colección de libros.
- **📅 Préstamos y Reservas**: Gestión eficiente de préstamos activos ("My Loans").
- **⚡ Rendimiento Optimizado**: Gracias a **Nix.js**, la aplicación ofrece una experiencia de usuario fluida con una carga mínima y reactividad instantánea en la interfaz.
- **🎨 Diseño Moderno**: Interfaz limpia y responsiva construida con Tailwind CSS.

## 🛠️ Instalación y Uso Local

Para ejecutar este proyecto localmente, asegúrate de tener [Bun](https://bun.sh) instalado:

1. **Clona el repositorio:**
   ```bash
   git clone https://github.com/DeijoseDevelop/parcial-1-analisis.git
   cd parcial-1/frontend
   ```

2. **Instala las dependencias:**
   ```bash
   bun install
   ```

3. **Inicia el servidor de desarrollo:**
   ```bash
   bun dev
   ```

4. **Abre en tu navegador:**
   Visita `http://localhost:5173` (o el puerto indicado en la consola).
